# 期货行情 PWA

这是一个可安装到电脑和手机的 PWA 应用，用来查看上海期货交易所沪铝、螺纹钢期货行情。

功能：

- 查看沪铝、螺纹钢连续及近月合约报价。
- 支持品类选择：沪铝、螺纹钢。
- 点击任意合约，查看该合约 K 线图。
- 支持 1小时、3小时、5小时、日线、周线、月线。
- 小时线底部显示更密的日期和具体时刻坐标。
- 支持 60、120、240、全部 K 线显示数量。
- 支持显示一条 MA 均线，可选空、MA5、MA10、MA20、MA30。
- 支持叠加策略：空、更新1、-1Best、量价；策略名称会跟随品类显示为铝或螺纹钢。
- 支持 K 线图白底/黑底切换。

## 运行

在 PowerShell 中进入本目录，然后运行：

```powershell
.\start-app.ps1
```

打开：

```text
http://localhost:8787
```

电脑浏览器可直接安装。手机端如果要像原生 App 一样安装，建议部署到 HTTPS 域名后再用手机浏览器打开并添加到主屏幕。

## 临时公网 HTTPS 部署

运行：

```powershell
.\start-public.ps1
```

脚本会打印一个 `https://...trycloudflare.com/?v=14` 地址。手机或其他电脑打开这个地址即可访问。

这是临时地址，关闭电脑、关闭本地服务或关闭 Cloudflare Tunnel 后就会失效；下次运行脚本会生成新的地址。

## 数据说明

- 行情：新浪财经公开期货行情接口。
- K 线：新浪财经公开期货 K 线接口。
- 合约规格：上海期货交易所对应期货品种说明。
- 显示单位：元/吨；交易单位：5 吨/手。

行情数据可能延迟或短暂不可用，交易决策请以交易所及券商终端为准。
